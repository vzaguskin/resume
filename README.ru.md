## Опыт работы

### МТС AI — Руководитель центра компетенций Voice ML

**Сентябрь 2022 — настоящее время** · 5 команд, 33 человека · Remote

Руковожу полным циклом разработки машинного обучения и speech-технологий для ключевых продуктов МТС: виртуальный ассистент, голосовой бот «Секретарь», интеграция с Wildberries. Управляю тимлидами, ставлю цели, провожу performance review.

**Ключевые результаты:**
- Снизил WER на 3-5 п.п. на всех ключевых продуктах (ассистент, голосовой бот, секретарь) через оптимизацию model training и data pipeline
- Запустил в production: Target-Speech VAD (+1 п.п. WER), conversational TTS, voice cloning, keyword spotting, frame-level VAD, акустическую пунктуацию
- Построил ML-инфраструктуру: пайплайны model training, автоматическая разметка данных (auto-labeling), трекинг экспериментов, MLOps мониторинг
- Спроектировал и задеплоил realtime voice agent pipeline: EPD, semantic VAD, barge-in detection, LLM orchestration, streaming TTS — оптимизация end-to-end latency
- Разработал методологию оценки endpoint detection с tiered scoring, валидированную на Full-Duplex-Bench-v3
- Оптимизировал inference для production: стриминг, ONNX конверсия, TensorRT, mobile deployment

**Организационный вклад:**
- Сформировал и возглавил 5 кросс-функциональных команд (ASR, TTS, VAD/EPD, KWS, Infrastructure) с нуля
